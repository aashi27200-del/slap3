# 💖 Slap3 — Talking Aashi

## Deploy to GitHub Pages

### Step 1 — Create repo named `slap3`
1. Go to https://github.com → click **+** → **New repository**
2. Name it exactly: `slap3`
3. Set to **Public**
4. Click **Create repository** (do NOT add README)

### Step 2 — Upload via Git
Open terminal in this folder:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/slap3.git
git push -u origin main
```

### Step 3 — Enable GitHub Pages
1. In your repo → **Settings** → **Pages**
2. Under **Source** → select **GitHub Actions**
3. Wait ~1 minute ⏳

### Step 4 — Your site is live! 🎉
`https://YOUR_USERNAME.github.io/slap3`
